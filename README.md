# CodAlpha_Artifical-Intelligenc-Chatbot
CodeAlpha AI ChatBot

Project Description

This project is a Java-based Artificial Intelligence ChatBot developed using Java Swing. The chatbot provides a simple graphical user interface (GUI) where users can interact by typing messages. It uses rule-based logic to understand common questions and respond with appropriate answers. The application demonstrates Java programming, GUI development, and basic chatbot concepts.

Features

- Interactive Java Swing GUI
- Rule-based AI chatbot
- Greets users with friendly messages
- Responds to common questions
- Displays current date and time
- Tells programming jokes
- Easy-to-use interface
- Fast and lightweight application
- User-friendly chat window
- Real-time conversation using GUI

How to Run

1. Download or clone the repository.
2. Open the project in VS Code.
3. Compile the Java files.
4. Run "ChatBotGUI.java".
5. Type a message and click Send.
